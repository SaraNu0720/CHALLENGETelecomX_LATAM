# 📊 Telecom X - Análisis de Evasión de Clientes (Churn)

Este proyecto analiza la evasión de clientes (churn) en Telecom X utilizando Python y técnicas de análisis exploratorio de datos (EDA). El objetivo es identificar los factores que influyen en la cancelación del servicio para apoyar el desarrollo de estrategias de retención.

---

#  Objetivo del Proyecto

Comprender los factores asociados a la cancelación de clientes en Telecom X mediante análisis de datos y visualizaciones que permitan generar insights para reducir la tasa de churn.

---

#  Tecnologías Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

---

#  Dataset

Los datos fueron obtenidos desde una **API en formato JSON** que contiene información sobre clientes de Telecom X.

El dataset incluye variables como:

- Información demográfica
- Servicios contratados
- Tipo de contrato
- Método de pago
- Cargos mensuales
- Antigüedad del cliente
- Estado de cancelación (Churn)

---

#  Análisis Realizados

Durante el proyecto se realizaron las siguientes etapas:

### 1. Extracción de datos (ETL)
Se cargaron los datos desde la API y se transformaron en un DataFrame para su análisis.

### 2. Limpieza y preparación de datos
Se realizaron procesos de limpieza como:

- Conversión de variables categóricas
- Eliminación de inconsistencias
- Conversión de tipos de datos

### 3. Análisis Exploratorio de Datos (EDA)
Se analizaron diferentes variables para identificar patrones asociados al churn.

Los principales análisis incluyen:

- Distribución de clientes que cancelaron el servicio
- Churn según género
- Churn según tipo de contrato
- Churn según tipo de servicio de internet
- Relación entre cargos mensuales y churn
- Relación entre antigüedad del cliente y churn
- Matriz de correlación

---

#  Resultados

El análisis mostró que aproximadamente **26.54% de los clientes cancelaron el servicio**.

Los factores más relacionados con la evasión de clientes incluyen:

- Contratos **mes a mes**
- **Altos cargos mensuales**
- **Baja antigüedad del cliente**
- Servicio de **internet de fibra óptica**

---

#  Conclusiones

Los resultados sugieren que la empresa puede reducir la tasa de churn mediante:

- Incentivos para contratos de largo plazo
- Mejora del servicio de internet de fibra óptica
- Programas de retención para nuevos clientes
- Optimización de precios y planes de servicio
